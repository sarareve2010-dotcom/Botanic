# Botadex Expert

Application web statique/PWA pour apprendre botanique, EUNIS, phytosociologie, relevés et bioévaluation.

## GitHub Pages

Le dépôt contient les fichiers de l’application à la racine. Pour la mettre en ligne : Settings > Pages > Deploy from a branch > main > /root.

## Important

Données pédagogiques : vérifier les statuts officiels avec INPN/TAXREF, DREAL, CBN, OFB, EUNIS/HabRef avant tout rapport.
